# 💈 Sistema de Agendamento de Barbearia

Sistema web desenvolvido em PHP para gerenciamento de agendamentos de uma barbearia. O projeto permite que usuários escolham serviços, realizem agendamentos e acompanhem suas marcações de forma simples e organizada.

---

## 🚀 Funcionalidades

- Cadastro e login de usuários
- Sistema de sessão
- Listagem de serviços disponíveis
- Agendamento de serviços pelo usuário
- Visualização de agendamentos
- Cancelamento de agendamentos
- Área administrativa para gerenciamento de serviços
- Upload de imagens dos serviços

---

## 🛠️ Tecnologias utilizadas

- PHP
- HTML5
- CSS3
- JavaScript (básico)
- SQLite (banco de dados)
- Composer (autoload e organização do projeto)
